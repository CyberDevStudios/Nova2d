# Configuration

Nova2D has three configuration files at the project root.

---

## `conf.lua` — Love2D Configuration

Controls the window, loaded modules, and save directory.

```lua
function love.conf(t)
    t.identity = "nova2d"
    t.window.title = "Nova2D"
    t.window.width = 800
    t.window.height = 600
    t.window.resizable = true
    t.window.vsync = 1
    t.window.icon = "assets/images/icon32.png"
    t.console = false
    t.modules.audio = false
    t.modules.physics = false
    t.modules.joystick = false
end
```

### Fields

| Field | Type | Default | Description |
|---|---|---|---|
| `t.identity` | string | `"nova2d"` | Save directory (`love.filesystem.getSaveDirectory()`) |
| `t.window.title` | string | `"Nova2D"` | Window title |
| `t.window.width` | number | `800` | Window width in pixels |
| `t.window.height` | number | `600` | Window height in pixels |
| `t.window.resizable` | boolean | `true` | Allow window resizing |
| `t.window.vsync` | number | `1` | 1 = enabled, 0 = disabled |
| `t.window.icon` | string | — | Path to the window icon |
| `t.console` | boolean | `false` | Show debug console (Windows) |
| `t.modules.audio` | boolean | `false` | Load audio module (disabled by default in Nova2D) |
| `t.modules.physics` | boolean | `false` | Load Box2D physics module |
| `t.modules.joystick` | boolean | `false` | Load joystick module |

> `t.modules.audio` is disabled by default in Nova2D. If you need sound, change it
> to `true`. This reduces startup time and memory usage when you don't need audio.

> `t.modules.physics` is disabled — Nova2D doesn't use Box2D. Use `bump.lua` for
> AABB collisions (installable via `love gestor/ install bump.lua`).

**Example: project with audio:**

```lua
function love.conf(t)
    t.window.title = "My Game"
    t.window.width = 1024
    t.window.height = 768
    t.modules.audio = true  ← enable audio
end
```

## `nova2d.lua` — Project Manifest

Declares the project name, version, author, and dependencies.

```lua
return {
    name    = "my-game",
    version = "1.0.0",
    author  = "Your Name",

    dependencies = {
        ["bump.lua"] = {
            repo = "kikito/bump.lua",
            version = "3.1.7",
            type = "single",
            file = "bump.lua"
        },
    },
}
```

### Project Fields

| Field | Type | Required | Description |
|---|---|---|---|
| `name` | string | yes | Project name |
| `version` | string | yes | Semver version |
| `author` | string | no | Author or team |

### Dependency Fields

Each entry in `dependencies` uses the name as the key.

| Field | Type | Required | Description |
|---|---|---|---|
| `repo` | string | yes | `user/repo` on GitHub |
| `version` | string | yes | Tag, branch, or commit SHA |
| `type` | string | yes | `"single"` (one file) or `"multi"` (ZIP) |
| `file` | string | if type="single" | File name within the repo |

## `nova2d-lock.lua` — Lockfile

Automatically generated by the dependency manager. **Do not edit manually.**

```lua
-- Generated automatically. Do not edit.
return {
    ["bump.lua"] = { version = "3.1.7", installed = 1750617600 },
    ["anim8"]    = { version = "2.3.0", installed = 1750617600 },
}
```

Stores the exact version and installation timestamp for reproducible builds.
When you run `love gestor/ install`, the manager compares this file with
`nova2d.lua` and only downloads what's missing or changed.
