# Configuration

## conf.lua

The project's `conf.lua` configures Love2D window settings:

```lua
function love.conf(t)
    t.window.width = 800
    t.window.height = 600
    t.window.title = "Nova2D"
    t.window.resizable = true
    t.window.vsync = true
    t.version = "11.0"
end
```

You can change window size, title, resize behavior, and vsync here.
