# GitHub

The Nova2D source code and issue tracker are on GitHub.

## Repository

**New location**: [github.com/CyberDevStudios/Nova2d](https://github.com/CyberDevStudios/Nova2d)

## Contributing

1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Open a pull request

## Reporting issues

Open an issue on GitHub with:
- What you were doing
- What happened
- What you expected to happen
- Love2D version and operating system
